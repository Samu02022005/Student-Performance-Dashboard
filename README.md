# Student Performance Dashboard

## Project Overview

This project was developed using Microsoft Power BI to analyze student performance data. The dashboard provides insights into attendance percentage and average marks using interactive visualizations.

## Features

* KPI Cards (Total Students, Average Attendance, Average Marks)
* Attendance Distribution Pie Chart
* Student-wise Average Marks Bar Chart
* Student Details Table Visual
* Interactive Dashboard Design

## Dataset

The dataset contains:

* Student ID
* Student Name
* Attendance Percentage
* Average Marks

## Tools Used

* Microsoft Power BI
* Microsoft Excel
* Data Visualization Techniques
